Rodzaje Pętli w JavaScript

Pętla for

Pętla for służy do wykonania określonej liczby iteracji.

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Pętla while (kontrolowane wejście)

Pętla while sprawdza warunek przed wykonaniem iteracji.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Pętla do...while (kontrolowane wyjście)

Pętla do...while wykonuje kod przynajmniej raz, a potem sprawdza warunek.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
